Securiteam wargame 2018

Welcome everyone on this year's wargame!

Every new stage is unlocked by solving 3 challenges on the previous stage!

The solution of every assignment is a 32 characters long MD5 hash wrapped into "SECURITEAM{...}" (e.g. "SECURITEAM{0123456789abcdef0123456789abcdef}"). If you find the hash without the pre- and postfixes, then you should wrap it in yourself.

You can use the hint button on any challenge for a 50% penalty in points!
